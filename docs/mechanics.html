Character

Creation:


When you first enter the game, you'll be prompted to create a Character Profile, which allows you to customize the character you'll use while playing.


During character creation, you'll be able to customize the following:

  • Hair
  • Face & Facial Hair
  • Skin Tone
  • Starter Class

You can either select from the available options in the catalog (hair, face) or use your avatar's face accessories (hair, face, neck accessories, hats).


For the Starter Class, you will choose between two options:

  • Swordman
  • Berserker

The class you select only determines your starting class upon joining the game. You can still switch to the other class later by equipping its weapon type.


The starter class can only be selected once. After finalizing your profile, you wonâ€™t be able to edit it, though switching classes in-game is easy.

Finalizing your character profile will lock in your chosen appearance and class. However, you can modify your appearance later through Customization.

Customization:


To access customization, you will need one of the following:

  • Unlimited Customization Game-pass: A one-time purchase that grants unlimited customization access.
  • One-time Customization Ticket: Grants access for a single customization session.

If you use a ticket and cancel without making changes, the ticket is still consumed!
